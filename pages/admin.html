In-Place Page Editor
✎
Edit live page text directly

Click the button below to enter edit mode. Every heading, paragraph, and label on the current page will become clickable and editable — right on the page, exactly as it looks to visitors. When you're done, save to download the updated file and drop it into your /pages/ folder.

1Close this panel — navigate to the page you want to edit (Home, About, Books, etc.)
2Reopen Admin (Ctrl+Shift+A) → Edit Pages → turn on Edit Mode
3Close the panel again — every text element is now editable. Click and type.
4Open Admin again → Edit Pages → Save — the updated file downloads automatically.
ⓘ  Only plain text elements are editable — buttons, SVGs, calculators, and navigation are protected. Structural HTML is never touched.
