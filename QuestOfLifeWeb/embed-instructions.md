# Embed Quest of Life on Your Site

## Option 1: Iframe (Easiest - Full App)
```html
<iframe 
  src="https://your-domain.com/QuestOfLifeWeb/index.html" 
  width="100%" 
  height="800" 
  frameborder="0"
  style="border-radius: 12px; box-shadow: 0 8px 32px rgba(0,0,0,0.3);">
</iframe>
```

## Option 2: Local Copy (Self-Hosted)
1. Copy entire `QuestOfLifeWeb/` folder to your site
2. Link `<a href="/path/to/QuestOfLifeWeb/index.html" target="_blank">Play Quest of Life</a>`

## Option 3: Single-File Embed (Advanced)
Extract core JS/CSS into single HTML (ask if needed).

**Responsive**: Works on mobile/desktop. Save state in localStorage.

Ready-to-use RPG numerology app! 🚀
